## Instalación
**Requirimientos previos** -> node.js</br>
Instalar dependencias:
```npm
/fronted/reto_final/ npm install
```
Instalar Vite:
```npm
/fronted/reto_final/ npm install vite
```
Run:
```npm
/fronted/reto_final/ npm run dev
```
